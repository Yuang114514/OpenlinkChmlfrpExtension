# Openlink Chmlfrp Extension
An OpenLink addon for Chmlfrp
### Based on OpenLink's Frpc Interface
[See OpenLink project](https://github.com/StarCarefree/OpenLink)
[See Frpc docs](https://terry-mc.top/javadoc/fun/moystudio/openlink/frpc/package-summary.html)
### Feature

- Login & Logout
- Sensitive Information Storage using java's Prefences API, not in game Directory
- Proxy Create and Remove
- Frpc Management
